Overview

Welcome to the GeoLink API documentation. This API provides programmatic access to GeoLink's powerful location-based NFT platform, allowing you to manage wallet locations, track NFTs, and integrate geospatial analytics into your applications.

Whether you are a wallet provider looking to integrate location services for your users or a data consumer interested in geospatial NFT data, our API offers robust and secure endpoints to meet your needs.

API Access

Authentication

All API requests must include an API key in the X-API-Key header:

X-API-Key: YOUR_API_KEY

API keys are role-based and provide access to different endpoints based on user permissions.
