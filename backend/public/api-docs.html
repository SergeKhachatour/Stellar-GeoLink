Overview

Welcome to the GeoLink API documentation. This API provides programmatic access to GeoLink's powerful location-based platform, allowing you to manage wallet locations, track assets, and integrate geospatial analytics into your applications.

Whether you are a wallet provider looking to integrate location services for your users or a data consumer interested in geospatial data, our API offers robust and secure endpoints to meet your needs.

API Access

Authentication

All API requests must include an API key in the X-API-Key header:

X-API-Key: YOUR_API_KEY

API keys are role-based and provide access to specific endpoints based on your user type (Wallet Provider or Data Consumer).

Key Endpoints

POST /api/location/update
Update wallet location with latitude, longitude, and optional metadata
GET /api/nft/nearby
Find NFTs near a specific location within a radius
GET /api/wallet-provider/user-locations
Get all locations for a specific wallet provider
POST /api/wallet-provider/privacy-settings
Configure privacy settings for location data
POST /api/wallet-provider/visibility-settings
Configure visibility settings for location data

Rate Limiting

API requests are rate limited to ensure fair usage. Current limits:

  • General endpoints: 1000 requests per hour
  • Location updates: 100 requests per hour
  • NFT queries: 500 requests per hour
