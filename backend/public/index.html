🚀 Stellar GeoLink API

Welcome to the Stellar GeoLink API server. This backend provides location-based services for blockchain assets.

📚 API Documentation

📖 Swagger API Docs

🔗 Available Endpoints

  • Location Services: Find nearby wallets and NFTs
  • Wallet Provider APIs: Manage user locations and settings
  • Data Consumer APIs: Query wallet and NFT data
  • NFT Management: Pin, collect, and transfer NFTs

🔐 Authentication

This API uses API key authentication. Include your API key in the X-API-Key header.

🌍 Location Services

Interactive location-based services with coordinate validation and map integration.
